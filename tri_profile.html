Ideal Job: Unity Game Developer



Job description

For this job, the employee is responsible for creating games using Unity Engine – a popular game engine, widely used around the world. He is also in charge of operating the company’s developing application: App game, as well as building developing environments like: test server for testers to check for gameplays, bugs, and so on; application servers/hosts for managing and manipulating sources of data in the games. Another requirement for the employee is performing optimization to the servers to minimize update and maintenance tasks. Based on my understandings, this is done to cut off the expenses and reduce the rate of bugs appearing. Additionally, server optimization also aims to decrease overload while operating the system to improve the user experience. Finally, taking serious consideration into user feedback to improve the products’ quality is a critical aspect for this job. What interests me from this position is the chance of developing games using Unity Engin, along with developing application servers. These are all the factors that I’ve been very curious about and researched a lot on them, as I look forward to working more with them in the future.

  • Experience in Game Design, cartoons, and games.
  • Good use of English in work. Knowing Japanese is a huge plus.
